# linux常用命令

## mkdir 创建文件夹

```
mkdir 文件夹名
```
 
 -p 递归创建
```
 mkdir -p cc/ee
```

## touch 创建文件
```
touch 文件名
```

## echo 写入内容（将hello写入到index.txt）
```
echo hello > index.txt
```

- 追加内容

```
echo hello >> index.txt
```


## cat 查看文件内容
```
cat 文件名
```

## cd切换目录
```
cd 目录路径
```

- 切换到上级目录

```
cd ..
```

## ls 查看当目录下有哪些文件
```
   ls
```

- al 查看所有文件及文件信息

```
 ls -al
```


##  pwd查看当前所在目录路径
```
 pwd
```

## rm 递归删除
```
rm -rf 文件名
```
